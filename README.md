# django_project
Задание от Яндекс Практикума по вариантам.
Вариант 24. Парикмахерская.

В парикмахерской работают мастера, каждый из которых умеет оказывать определённый набор услуг. На оказание каждой из услуг каждому мастеру требуется своё время. Перечень предоставляемых парикмахерской услуг и цен описан в прейскуранте. Клиенты должны предварительно записаться на посещение. По факту посещения должны быть сохранён перечень оказанных услуг, стоимость посещения. У клиентов могут быть предпочтения по мастерам. У клиентов сайта должен быть функционал по назначению и отмене записей, у администратора – функционал по добавлению, редактированию и удалению перечня мастеров, услуг и цен.
