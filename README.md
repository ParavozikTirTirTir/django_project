# django_project
Задание от Яндекс Практикума по вариантам.
Вариант 24. Парикмахерская. В парикмахерской работают мастера, каждый из которых умеет оказывать определённый набор услуг. На оказание каждой из услуг каждому мастеру требуется своё время. Перечень предоставляемых парикмахерской услуг и цен описан в прейскуранте. Клиенты должны предварительно записаться на посещение. По факту посещения должны быть сохранён перечень оказанных услуг, стоимость посещения. У клиентов могут быть предпочтения по мастерам (не четкий пункт тз). У клиентов сайта должен быть функционал по назначению и отмене записей, у администратора – функционал по добавлению, редактированию и удалению перечня мастеров, услуг и цен.

Сайт поддерживает регистрацию клиентов, управление мастерами и услугами, запись на приём и админку.
## Основные роли клиента
- Регистрация и вход
- Просмотр мастеров и услуг
- Онлайн-запись с выбором мастера, услуг и времени
- Просмотр и отмена своих записей
## Основные роли администратора
- Управление мастерами (добавление, редактирование, скрытие)
- Настройка прайс-листа (услуги, цены, примерная длительность (но у каждого мастера она разная, это число лишь ориентировочное))
- Настройка индивидуального времени выполнения услуги для каждого мастера
- Подтверждение/отмена/завершение записей клиентов
- Просмотр и удаление зарегистрированных пользователей

## Быстрый запуск
1. Клонируйте репозиторий
- git clone <ваш-репозиторий>
- cd salon_project

3. Создайте виртуальное окружение
- python -m venv venv для Windows: 
- venv\Scripts\activate

5. Установите зависимости
- pip install -r requirements.txt

6. Примените миграции и загрузите демо-данные (если они загрузятся)
- python manage.py migrate
- python manage.py loaddata fixtures/salon.json

7. Создайте администратора
- python manage.py createsuperuser  # логин: admin, пароль: admin123

8. Запустите сервер
- python manage.py runserver
